## 5. 正确思路
1. 初始化边界
2. 循环条件
3. 计算middle
4. 比较收缩
5.结束处理 

## 6. 易错点
- [left,right]
  - 循环条件:`while (left <= right)`
  - 循环不变量的选择:`right` 要赋值为 `middle - 1`
- [left,right)
  - 循环条件:`while (left < right)`
  - 循环不变量的选择:`right` 要赋值为 `middle`

## 7. 复杂度
- 时间复杂度：O(log n)
- 空间复杂度：O(1)

## 9. 代码

```java
// 以[left,right]为例
class Solution {
    public int search(int[] nums, int target) {
        if(target < nums[0] || target > nums[nums.length - 1]){
            return - 1;
        }
//如果 target 小于数组最小值（nums[0]）或大于最大值（nums[nums.length - 1]），则目标值一定不存在。
//直接返回 -1，避免进入后续的循环计算。
//将这些特定情况的时间复杂度从 O(logN) 降低到 O(1)。
        int left = 0, right = nums.length - 1;
        while(left <= right){
            int mid = left + ((right - left) >> 1); //位运算的写法，本质等同于 left + (right - left) / 2
            if(nums[mid] == target){
                return mid;
            }
            else if (nums[mid] < target){
                left = mid + 1;
            }
            else{
                right = mid -1;
            }
        }
        return -1;


    }
}
```
